## 💻 Frontend Setup

### Requirements

- Node.js 20+

- IDE (e.g., VSCode)

### Steps

1. Fork and clone the frontend repository: [ToDo Front](https://github.com/mehrdad-javan/todo-front)

2. Run the React frontend.

3. Check existing routes, navigation, and context setup.
